Cette action vérifie si l\textquotesingle{}utilisateur a accès. Si l\textquotesingle{}accès est accordé, elle récupère le rôle de l\textquotesingle{}utilisateur. Ensuite, elle obtient le modèle pour interagir avec les données. Elle récupère l\textquotesingle{}ID du formateur à partir des paramètres GET. Elle récupère les détails du formateur par ID à partir du modèle. Elle récupère les informations supplémentaires du formateur à partir du modèle. elle affiche la vue \textquotesingle{}formateurs\+\_\+details\textquotesingle{} avec les données.
