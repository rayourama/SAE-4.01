action\+\_\+list (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Retourne une liste de discussions pour l\textquotesingle{}utilisateur authentifié.

Cette méthode récupère toutes les discussions de l\textquotesingle{}utilisateur, puis parcoure chaque discussion pour obtenir les détails nécessaires. Les informations de chaque discussion, telles que le nom de l\textquotesingle{}interlocuteur, le nombre de messages non lus, et l\textquotesingle{}URL de la discussion, sont stockées dans un tableau. Ce tableau est ensuite utilisé pour afficher une liste de discussions dans la vue \textquotesingle{}discussion\+\_\+list\textquotesingle{}.
