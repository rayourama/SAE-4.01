action\+\_\+register (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}

Inscription d\textquotesingle{}un nouvel utilisateur.

Cette fonction vérifie si la requête est une requête POST et vérifie que les champs requis ne sont pas vides. Elle nettoie ensuite les valeurs des champs et vérifie les longueurs des champs. Elle vérifie également si l\textquotesingle{}email a un format valide et si le nom et le prénom ne contiennent que des lettres et des tirets. Si le rôle de l\textquotesingle{}utilisateur est valide, elle crée un nouvel utilisateur et génère un lien de vérification envoyé par email. Si la création de l\textquotesingle{}utilisateur réussit, elle affiche un message de succès et envoie un email de vérification. Sinon, elle affiche un message d\textquotesingle{}erreur. Si la requête n\textquotesingle{}est pas une requête POST, elle affiche un message d\textquotesingle{}erreur. Enfin, elle rend la vue "{}auth"{} sans données spécifiques.
