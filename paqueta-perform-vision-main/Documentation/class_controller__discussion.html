◆ action_list()

action_list ( )

Retourne une liste de discussions pour l'utilisateur authentifié.

Cette méthode récupère toutes les discussions de l'utilisateur, puis parcoure chaque discussion pour obtenir les détails nécessaires. Les informations de chaque discussion, telles que le nom de l'interlocuteur, le nombre de messages non lus, et l'URL de la discussion, sont stockées dans un tableau. Ce tableau est ensuite utilisé pour afficher une liste de discussions dans la vue 'discussion_list'.
